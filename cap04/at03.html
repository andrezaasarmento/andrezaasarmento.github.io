Aqui está um roteiro inicial para você, novo aluno, se guiar mais facilmente em seus estudos:

  1. Lógica de Programação e Algoritmos
    • Entenda como o computador pensa. Estude variáveis, estruturas condicionais (se/senão) e laços de repetição (loops).
  2. Matemática Discreta ou Lógica Computacional
    • É a base teórica para a computação. Foque em tabelas verdade, teoria dos conjuntos e grafos, essenciais para o desenvolvimento futuro de softwares.
  3. Introdução aos Sistemas de Informação
    • Familiarize-se com os conceitos de hardware, software, redes e banco de dados. Aprenda sobre o ecossistema corporativo e como a TI resolve problemas organizacionais.

Para ter uma base mais consolidada os requisistos abaixo são recomendados:

  1. Fundamentos de Arquitetura de Computadores
    • Descubra o que acontece "por baixo do capô". Estude como o processador, a memória e os sistemas operacionais funcionam e interagem.
  2. Gestão e Administração Básica
    • Sistemas de Informação unem tecnologia e negócios. Entenda os conceitos iniciais de processos empresariais e como os dados apoiam a tomada de decisão.
  3. Prática Diária e Resolução de Problemas
    • Crie uma rotina diária de estudos de pelo menos 1 hora para colocar a teoria na prática, resolvendo exercícios de lógica e escrevendo códigos simples.

Top 5 tecnologias

Aqui você vai conhecer as top 5 tecnologias mais utilizadas no ramo de Sistemas da Informação

  1. Vue.js
    • Um framework progressivo criado por Evan You. Ele é conhecido por sua curva de aprendizado mais suave e flexibilidade, unindo boas ideias encontradas no React e no Angular.
  2. Angular
    • Um framework robusto, estruturado e completo mantido pelo Google. Ele dita um padrão rígido de arquitetura e é amplamente utilizado em sistemas corporativos de grande escala.
  3. TypeScript
    • Um "superset" (superconjunto) do JavaScript criado pela Microsoft. Adiciona tipagem estática opcional, o que ajuda a encontrar erros enquanto você digita o código antes de executá-lo.
  4. React
    • Uma biblioteca de código aberto mantida pela Meta (Facebook) focada em construir interfaces de usuário (UI). Usa uma arquitetura baseada em componentes reutilizáveis e o conceito de Virtual DOM para otimizar a renderização.
  5. JavaScript
    • A linguagem de programação central da web. Permite criar páginas interativas e dinâmicas, executada nativamente em todos os navegadores.

Glossário de termos técnicos

Estes são alguns termos técnicos que são fundamentais para um bom desenvolvimento inicial como desenvolvedor

HTML
É o "esqueleto". Define a estrutura e o conteúdo da página, organizando textos, imagens e formulários através de marcações chamadas tags.
CSS
É a "roupa" e a maquiagem. Responsável pelo design, layout, cores, fontes e responsividade, garantindo que a página fique bonita e se adapte a celulares ou computadores.
JavaScript
É o "cérebro" e os músculos. Adiciona dinamismo e interatividade. Permite validar campos, criar animações, abrir menus laterais e se comunicar com o servidor sem precisar recarregar a página.
Funciona como a ferramenta que altera o DOM. O JavaScript pode criar, modificar ou excluir elementos HTML e estilos CSS dinamicamente.
DOM
Document Object Model, ou Modelo de Objeto do Documento
É uma representação em árvore da página HTML que o navegador cria. O DOM transforma todo o código em "objetos" que podem ser facilmente lidos e alterados pelo JavaScript, permitindo mudar qualquer coisa na página em tempo real.
API
É a ponte de comunicação. Um conjunto de regras que permite que diferentes sistemas ou partes de um sistema conversem entre si, buscando ou enviando dados (por exemplo, quando um site busca a previsão do tempo).
HTTP
HTTPS
É a linguagem padrão que permite a comunicação na web. Define como os navegadores (clientes) enviam pedidos e como os servidores devolvem os dados (páginas, imagens, etc.).
URI
URL
É o endereço web. É o que você digita na barra de endereços do navegador (ou clica) para acessar um recurso específico na internet. Exemplo: https://google.com.br.
DNS
Sistema de nomes de domínio; traduz nomes de domínio legíveis por humanos em endereços IP numéricos.
